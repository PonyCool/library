# PonyCool Library是什么

PonyCool Library 是一个为Codeigniter4框架提供支持的扩展库

# 要求

- PHP 8.2 或更高版本
- Codeigniter 4.4.0 或更高版本

# 使用

```
composer require ponycool/ponycool-library
```

# 测试

```
make test
# 手动测试单个文件
phpunit tests/File
```
